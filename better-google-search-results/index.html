UnFuck Google-Search Shitty-Results By Actually Pre-Removing / Pre-Avoiding Many Known-To-Be-Crappy Websites!

Posted at Sat, 27 May 2017 12:51:31 GMT



For Google-Chrome.


Go ahead and enter the following URL: chrome://settings/searchEngines

and add new item with the name unfucked_google_search (or anything else..) and the value of:
https://www.google.co.il/search?num=50&q=%s%20-site%3A*.w3schools.com%20-site%3A*.yammer.com%20-site%3A*.f5.com%20-site%3Awww.daniweb.com%20-site%3A*.outbrain.com%20-site%3A*.taboola.com%20-site%3A*.visualstudio.com%20-site%3A*.tumblr.com%20-site%3A*.facebook.com%20-site%3Ahe-il.facebook.com%20-site%3A*.pictaram.com%20-site%3A*.twitter.com%20-site%3A*.instagram.com%20-site%3Awww.wikihow.com%20-site%3Awww.quora.com%20-site%3A*.stumbleupon.com%20-site%3Awww.pinterest.com%20-site%3A*.linkedin.com%20-site%3Aplus.google.com%20-site%3A*.flickr.com%20-site%3Abugzilla.mozilla.org%20-site%3Ajsfiddle.net%20-site%3Acodepen.io%20-site%3Abooks.google.com%20-site%3Awww.safaribooksonline.com%20-site%3A*.w3.org%20-site%3Awww.wufoo.com%20-site%3Awww.htmlcodes.ws%20-site%3A*.gnu.org

once you've done you simply hit the [ENTER] key, and hover the mouse over, and select make default.

You probably want to replace the google.co.il with google.com (or google.co.uk).

This is the minimal search URL that, will remove the majority of shitty results. Crafted for best results for an engineer/programmer.

feel free to add more stuff like this %20-site%3A*.yourdomain.com if you want to remove any more search-results, by shitty domains.

if you know the use of the domain, it is always better to avoid the * usage, for example you can safely use www. for quora.com since it only uses that sub-domain, but tumblr.com and stumbleupon.com often make use of the sub-domain for categories or users identification, so you probably want to use the *. for an effective blocking.


This might be showing you google captcha once since you're using some advance/non-standard search-engine query, keep the google cookies in your browser if you're using some clean-software such as CCleaner and it won't be too frequent..
