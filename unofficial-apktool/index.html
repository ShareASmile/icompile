apktool-cli-all_2.4.2-0143dc-SNAPSHOT.jar.zip



Notes:

  • 2.3.4

  • https://ibotpeaches.github.io/Apktool

  • Pre clear-up old-framework libs (will be recreated) by deleting the entire folder: C:\\Users\\USERNAME\\apktool\\.

  • after clean and update, use call "gradlew.bat" "build" "shadowJar" -x "test" to quick-build the JAR file without running any pre-tests, then from within the APKTool folder run explorer ".\brut.apktool\apktool-cli\build\libs" to quickly arrive to the location of the newly created JAR.
    The latest version of smali lib is in /smali.properties and the latest version of the baksmali lib is in /baksmali.properties in the JAR root (use 7zip, WinRAR or WinZIP to view the content..).
    The build-version is in /properties/apktool.properties, I usually publish the JAR with the version appended to the end of the filename apktool-cli-lib.
    To use it I usually rename it to simply apktool.jar.

  • ApkTool on XDA: forum.xda-developers.com/showthread.php?t=1755243
