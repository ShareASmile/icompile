You Should Disable Chrome's Native-Client

Posted at Thu, 23 Jun 2016 01:11:14 GMT



Native Client is a not-so-little parsing engine for "native code",
meaning not JavaScript but more closely to C++.

Go to chrome://plugins/ and disable the Native Client.
Making your browser safer (reduce functionality) and faster (no need to boot-load the engine).
