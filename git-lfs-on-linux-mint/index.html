Posted at Sun, 11 Jun 2017 18:31:40 GMT









  • Start with entering into higher-permission state using su -.

    It will make the following commands shorter,
    and this way you won't need sudo,

    it will also make sure internal-commands in the sh-file (below) will succeed,
    otherwise they will fail.. silently.. :/


  • Follow:
    Solved: Linux Mint And Debian Packages (https://icompile.eladkarako.com/solved-linux-mint-and-debian-packages/),
