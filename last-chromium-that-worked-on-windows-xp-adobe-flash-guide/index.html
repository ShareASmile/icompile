you can look in icompile.eladkarako.com for more information about the command line,
but the end result should look like this:
"C:\\Documents and Settings\\Elad\\Local Settings\\Application Data\\Chromium\\Application\\chrome.exe" --disable-bundled-ppapi-flash --ppapi-flash-path="C:\\Windows\\System32\\Macromed\\Flash\\PEPFLA~1.DLL" --allow-outdated-plugins --ppapi-flash-version=20.0.0.306


you probably don't need to change anything, since the short 8.3 dos format of the DLL naming, will match probably every version of the dll.
you might want to specify the flash version to help the Chrome logs and debugging be more accurate - it does not really matter.

you can look for the DLL file under the windows\\sy...\\pepf... .dll right click and check the attributes to see the version
