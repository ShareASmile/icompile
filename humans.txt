icompile.eladkarako.com - A Blog By Elad Karako. NodeJS cli. generated from an old WordPress SQL into a static HTML files, with a folder-structure that simulates WordPress' title-based permalinks. The website is hosted on GitHub, and the content of each article is written in a README.md file, which means each article can be viewed in the github repository https://github.com/eladkarako/icompile --- the blog contains a comment section for each article, based on GitHub-issues created by https://utteranc.es/ which works great :)
