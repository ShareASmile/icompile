Eclipse Kepler and lite configurations

Posted at Mon, 01 Aug 2016 23:36:28 GMT

Install Java7

Download at http://www.eclipse.org/downloads/index-developer.php (to the time written, Eclipse Kepler version 4.3 M5)
Extract the zip

Edit eclipse.ini  :  here is my eclipse.ini content:
-startup

plugins/org.eclipse.equinox.launcher_1.3.0.v20130125-144918.jar

--launcher.library

plugins/org.eclipse.equinox.launcher.win32.win32.x86_64_1.1.200.v20130117-173747

-product

org.eclipse.epp.package.jee.product

--launcher.defaultAction

openFile

--launcher.XXMaxPermSize

512m

-showsplash

org.eclipse.platform

--launcher.XXMaxPermSize

512m

--launcher.defaultAction

openFile

--launcher.appendVmargs

-vmargs

-XX:+UnlockExperimentalVMOptions

-XX:+UseG1GC

-XX:ParallelGCThreads=4

-XX:ConcGCThreads=4

-Xnoclassgc 

-Dsun.java2d.opengl=true

-Dosgi.requiredJavaVersion=1.7

-Dhelp.lucene.tokenizer=standard

-Xmn2g

-Xms4g

-Xmx4g

-Xss32m

-Xverify:none

set the place of eclipse.exe in environment-settings (ECLIPSE_HOME)


add "jetty run" and Velocity suppurt,
and flook (an eclipse-plugin I've made a year ago, shorthand of "file lookup")

https://github.com/eladkarako/icompile/releases/download/latest/Flook100201207021638.zip


(mirror#1, mirror#2)

browse folder:  eclipse/plugins/org.eclipse.platform_4.3.0.v201302041400/       (the _4.3.0.v....  will have another value probably..)

and edit the file splash.bmp in any image-editor client (Paint.Net)
