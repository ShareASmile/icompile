- As you can see the <code>N</code> in the <strong>"N-State"</strong> is set by modulus-operation, which can (also) be set as an attribute (for example <code>data-state-max</code>).

The dynamic-<strong>content</strong> is set using a little CSS trick,
in which you use the <code>:before</code> together with <code>attr</code> and <code>content</code>, this saves using JavaScript to change the content too....


This can be applied to any element, but it will work best with non-state elements (for example stuff that are not checkbox or select/option, since it will be very confusing and really bad UI-design practice).
