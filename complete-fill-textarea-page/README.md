The base of numerous of custom HTML-components 
and functional pages is a "textarea page" which consist of
a (surprise surprise..) <code>&lt;textarea&gt;</code> element
and some sort of functionality integrated by a form-submit, JavaScript,
or a mix of both.

<!--more-->

Although quite easy to manufacture- 
- a minimal, properly formatted template is 
  always nice to have handy on demand.

place in a separate HTML file and open it using JavaScript <code>window.open</code> to create a pop-up input-box,
place it in an iframe, use it as a <em>modal</em> component,
or simply use an a handy Unicode-supported editor (you browser is very good at <strong>presenting</strong> a wide variety of characters you probably won't find in any native OS-editor).
