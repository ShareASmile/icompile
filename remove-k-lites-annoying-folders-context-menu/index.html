Remove K-Lite's Annoying Folder's Context-Menu

Posted at Sat, 26 Nov 2016 18:25:43 GMT

Copy and paste the code(s) into .reg files, and run it.

Windows Registry Editor Version 5.00



[-HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.play]



[-HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.enqueue]


To undo:
Windows Registry Editor Version 5.00



[HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.play]

@="Play with MPC-HC"

"Icon"="\\"C:\\\\Program Files (x86)\\\\K-Lite Codec Pack\\\\MPC-HC64\\\\mpc-hc64.exe\\",0"



[HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.play\\command]

@="\\"C:\\\\Program Files (x86)\\\\K-Lite Codec Pack\\\\MPC-HC64\\\\mpc-hc64.exe\\" \\"%1\\""



[HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.enqueue]

@="Add to MPC-HC Playlist"

"Icon"="\\"C:\\\\Program Files (x86)\\\\K-Lite Codec Pack\\\\MPC-HC64\\\\mpc-hc64.exe\\",0"



[HKEY_CLASSES_ROOT\\Directory\\shell\\mplayerc64.enqueue\\command]

@="\\"C:\\\\Program Files (x86)\\\\K-Lite Codec Pack\\\\MPC-HC64\\\\mpc-hc64.exe\\" /add \\"%1\\""
