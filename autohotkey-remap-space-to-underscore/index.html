AutoHotKey Script - Replace (Re-Map) Space to Underscore

Posted at Wed, 17 Sep 2014 06:31:43 GMT

- having to work with a lot of MS-DOS or in environment that is not friendly to spaces in file-names?
- writing long sentences, and hitting multiple times the [SHIFT]+[-], which is result with unnatural-writing pattern?

the following script runs in the background, replacing the [SPACE] key with [_] (underscore), providing a continues writing.


;09:22 17/09/2014
;# name:         space_to_underscore.
;# description:  replace [SPACE] key with [_] key.

Space::Send _

;+Space::Send _                 ---- this will replace [SHIFT]+[SPACE] to [_]


;# components explained
;+       [SHIFT] key
;Space   [SPACE] key
;::Send  remap keys
;_       underscore



its AutoHotKey, so compiling it will result with an icon near the clock area (on your right, in LTR Windows OS),
you can disable/enable it when needed.
