Use MathType with WordPress

Posted at Sat, 26 Feb 2011 18:32:57 GMT

this is a short "1,2,3 guide" to help you post textual-content with an inline scientific formulas, in WordPress.

1. Download & Install: Latex for WordPress Plugin.

2. Open MathType.
?references -> Cut and Copy ?references...


select MathML or TeX:.
select Plain TeX from the box beneath.
uncheck Include translator name in translation
uncheck Include MathType data in translation


3. testing:
Create a new formula in MathType, Select it all, and Copy it.

Create a new post and just paste it anywhere, (you might want to use the HTML-post-editing), for example if you type:



you will get: Linear Equations ${p_1}(t){y^{(n - 1)}}$ are cool!
