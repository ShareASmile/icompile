Posted at Thu, 15 Jun 2017 12:53:56 GMT

Does This Looks Familiar?





You've probably followed instructions from:


  • https://deb.debian.org/
    https://backports.debian.org/Instructions/



  • Or even:




And, you've probably ran those lines:

  • su -

  • echo 'deb http://http.debian.net/debian wheezy-backports main' > /etc/apt/sources.list.d/wheezy-backports-main.list



Here is a solution/workaround/magic-trick
to resolve this pesky little issue:

Run this, (while still under su): gedit /etc/apt/sources.list.d/wheezy-backports-main.list
and replace the content with this one:
deb https://deb.debian.org/debian wheezy-backports main
deb https://deb.debian.org/debian stable main
deb https://deb.debian.org/debian-security stable/updates main


Save (of course), quit gedit,
and run this line(while still under su): aptitude install debian-archive-keyring.

Finally run (while still under su): apt-get update.
