New WordPress Plugin - TagClass-B-Gone

Posted at Tue, 02 Aug 2016 13:54:04 GMT

Do you have a lot of TAGs set to your post?
Well WordPress will add a class for each one to your post class-list,
effectively filling your page with a LOT of junk in form of tag(dash)*****



I find it useless.

So I've wrote a WordPress plugin to fix-it.




This home-page just saved itself 77,896 characters (about ~77KB!!!)

you can download it from the GitHub repository

and install it manually at your blog, should work on every-version since it uses raw HTML editing :]
